## Asteroids:

* A recreation of a timeless classic using PyGame.
---
### Prerequisites:

* Python3.13
* uv
* pygame 2.6.1
---

## Instructions:

* Install Python3 and uv
```
sudo apt update

sudo apt install -y python3

pip install uv
```
* Pin Python 3.13 with `uv`
```
uv python pin 3.13
```
* Clone the repository
```
git clone https://github.com/gym-halpert/asteroids-game
```
* Install requirements
```
uv pip install -r requirements.txt
```
* Run the game
```
python main.py
```
---
### Controls:
```
W = Forward
A = Rotate Left
S = Reverse
D = Rotate Right
SPACE = Shoot
```
